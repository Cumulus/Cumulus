Build Requirements
==================

 * OCaml >= 4.01
 * Eliom >= dev
 * Lwt >= 2.4
 * OCaml-safepass >= 1.2
 * PG'OCaml >= dev
 * MaCaQue >= dev
 * OCaml-markdown >= 0.1.1
 * Camomile
 * sqlite3-ocaml

Run Requirements
================

 * Ocsigenserver
 * PostgreSQL

See the INSTALL file to know how to get them.

A running demo is available at: http://cumulus.mirai.fr/
