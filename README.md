Build Requirements
==================

 * OCaml >= 4.01
 * Eliom >= 4.0
 * Lwt >= 2.4
 * OCaml-safepass >= 1.2
 * PG'OCaml >= 2.0
 * MaCaQue >= dev
 * OCaml-markdown >= dev
 * Batteries >= 2.2
 * Camomile

Run Requirements
================

 * Ocsigenserver with dbm
 * PostgreSQL

See the INSTALL file to know how to get them.

A running demo is available at: http://cumulus.mirai.fr/
